# KarimSaoud

Repository personale per consultare online e offline il sito e i progetti collegati.

## Consultazione online

- Repository GitHub: https://github.com/KarimSaoud/KarimSaoud
- Se GitHub Pages e' attivo dalla branch `main`, il sito statico puo' essere consultato da: https://karimsaoud.github.io/KarimSaoud/

## Consultazione offline

Per consultare i file offline:

1. Scarica il repository da GitHub con **Code > Download ZIP** oppure clona il progetto:

   ```bash
   git clone https://github.com/KarimSaoud/KarimSaoud.git
   ```

2. Apri `index.html` nel browser dalla cartella scaricata o clonata.

Le sottocartelle `projects/app-software/*` contengono anche progetti applicativi con dipendenze proprie.
